# Calculadora Avanzada

Calculadora web moderna con historial de resultados reutilizables, preparada para conversión a iOS nativa.

## Características

- ✅ Historial de resultados reutilizables
- ✅ Diseño responsive mobile-first
- ✅ Modo oscuro por defecto
- ✅ Animaciones fluidas y feedback táctil
- ✅ Evaluación matemática segura con math.js
- ✅ Preparada para conversión a React Native (iOS)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build para producción

```bash
npm run build
```

## Tecnologías

- React 18 + TypeScript
- Vite
- math.js (evaluación segura)
- CSS Variables (tema oscuro)
- CSS Grid + Flexbox (responsive)

## Conversión a iOS

Esta aplicación está estructurada para facilitar la conversión a React Native:

1. Los componentes están separados y son reutilizables
2. La lógica de negocio está en utilidades independientes
3. Los estilos usan variables CSS que pueden mapearse a StyleSheet de React Native
